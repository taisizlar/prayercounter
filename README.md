A simple, lightweight and accessible prayer counter (Tasbih) for Android.

## Features
- Count prayers with a single tap
- **Long press** the Add button to set a custom step — each tap will add that amount
- Save your count for later
- Works on lock screen
- Dark mode support (follows system theme)
- No unnecessary permissions — only vibration
- RTL layout support
